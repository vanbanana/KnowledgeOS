# KnowFlow 1.0 桌面版发布清单

## 已落地

- 安装包产物：`MSI` + `NSIS`。
- 打包版本：`1.0.0`。
- 图标策略：以 `SVG` 为单一源文件，自动生成各平台所需图标文件。
- parser 运行资源：只打包运行必需文件，不带测试目录和缓存文件。
- 数据目录策略：
  - 开发态默认写入仓库 `.knowledgeos`。
  - 发布态默认写入可执行文件目录下 `.knowledgeos`。
- 发布校验：新增“测试数据污染检查”脚本。

## 发布前必须确认

- 安装路径可写权限（发布态默认写安装目录下 `.knowledgeos`）。
- 首次启动配置引导（模型 API Key 为空时的提示流程）。
- parser 运行环境（目标机器 Python 3 与依赖是否满足）。
- 安装包签名（证书、时间戳、验签流程）。
- 升级策略（覆盖安装是否保留数据目录、版本回滚策略）。
- 卸载策略（卸载时是否保留用户导入资料与数据库）。

## 建议补充

- 崩溃与日志收集说明（日志目录、用户反馈入口）。
- 隐私与本地数据声明（数据存储位置、是否联网、传输范围）。
- 最小系统要求（Windows 版本、WebView2、磁盘与内存基线）。
- 冒烟测试脚本化（安装、首启、导入、图谱、搜索、导出演示文稿）。

## 可选环境变量

- `KNOWLEDGEOS_DATA_MODE=workspace|install|portable|appdata`
- `KNOWLEDGEOS_DATA_DIR=<自定义数据目录>`
- `KNOWLEDGEOS_PARSER_WORKER_PATH=<自定义 parser main.py 路径>`
